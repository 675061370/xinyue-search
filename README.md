## 更新日志

### v3.0 

- 新增 项目安装向导，简化部署流程

- 优化 解决部分数据导致的路由报错
- 优化 AI重构转存功能，提升稳定性与效率

- 增加网页全网搜功能
- 对接微信对话功能
- 更新优化Seo
- 后台增加批量删除功能
- 优化表格导入功能

- UI改版：不再使用uniapp
